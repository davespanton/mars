Mars
====

Trundling around Mars.

## Setup

Install [Elixir][elixir-install]
[elixir-install]: http://elixir-lang.org/install.html

```sh
# Get the deps
mix deps.get

# Run the tests
mix test

# Run the linter
mix dogma

# Run the test watcher
mix test.watch

# Explore a tiny planet
mix run scripts/main.exs
```
